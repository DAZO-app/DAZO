# DAZO — Decision At Zero Objection

DAZO is an open-source decision-making platform based on consent (inspired by sociocracy).

## 🎯 Purpose

DAZO helps groups make decisions efficiently by ensuring that proposals are adopted only when no valid objections remain.

It is designed for:

* Teams
* Communities
* Organizations
* Open-source projects

---

## 🧠 Core Concepts

* **Decision lifecycle** (draft → clarification → reaction → objection → revision → adopted)
* **Circles** (groups of users)
* **Feedback** (objections & suggestions unified)
* **Votes**
* **Roles & permissions**
* **Contextual participation**
